# Retail Sales Analytics Dashboard

## 📊 Project Overview
A Power BI dashboard analyzing retail sales 
data across customers, products and vendors.

## 📌 Key Metrics
- Total Sales: 113K
- Total Customers: 30
- Transactions: 465
- Total Products: 10
- Total Vendors: 3

## 💡 Key Insights
- Electronics is top category by sales
- V01 is top performing vendor
- Cash is most used payment method
- Chloe Carter is top customer
